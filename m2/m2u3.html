What is a Secure Programming Language?
What factors determine whether a programming language is secure or not?
Could Python be classed as a secure language? Justify your answer.
Python would be a better language to create operating systems than C. Discuss.
Response: How safe a programming language is depending on things like how it handles memory, its rules for using different types of data, and how well its built-in tools work. Python is usually considered safe because it does a good job with memory and has clear rules for data. But no language is completely safe. Python is great for many things, but it’s not the best choice for making operating systems because it’s not fast enough and doesn’t give you as much control over the computer’s hardware. C is better for these things because it’s faster and gives you more control.
